module:luhn-string

This module exports a factory for objects that generate alphanumeric strings that end with a checksum "digit". Digits in the string are in the range 0-9B-DF-HJ-NP-TV-XZ. All uppercase. To use '012345' as your sequence rather than this, simply create a new object with: 

var obj = new require('luhn-string')('012345');
