LuhnString

luhn-string. LuhnString

new LuhnString(validChars)

Represents a Luhn String generator and checker.

Parameters:
Name Type Description
validChars string

The valid characters for our luhn string objects, in sequence. Luhn Objects created with '0123456789' will return strings that do not validate with those created using the sequence '1234567890'.

In case you do not supply characters, a LuhnString will be created that uses the default sequence: '0123456789BCDFGHJKLMNPQRSTVWXZ'.
